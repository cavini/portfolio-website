The Movie App

Full Stack Web App that returns a list of movies based on user input and organizes it based on user preference, it can also return a random movie. Front end was built using React with the Context API for state management. The requests are made to a NodeJS app that uses Express middleware, routes and handlers to get the info from The Movie DB's API and returns the result to the user.
